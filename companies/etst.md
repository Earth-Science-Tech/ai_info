# Earth Science Tech, Inc. (ETST)

## Overview

Parent company that owns and operates multiple subsidiary businesses in the pharmacy and healthcare technology space.

**Ticker:** ETST (OTC)
**Primary Project:** eMed — electronic medical records and telemedicine platform

## Subsidiaries

| Company | Domain | Description |
|---------|--------|-------------|
| Rx Compound Store (RXCS) | rxcs | Compounding pharmacy |
| Mister Meds (MMed) | mmed | Pharmacy services |
| Peaks Curative | peaks | Curative e-commerce (client of eMed) |

## Engineering Team

Growing team of engineers using Claude AI for development. Shared knowledge managed through the `ai_info` repository.

- **Web Application:** Node.js / Express / EJS (emed_app)
- **ETL/Data:** Python / Prefect (emed_etl)
- **Database:** Azure SQL Server
- **Deployment:** Azure App Service
- **Source Control:** GitHub (Earth-Science-Tech org)
- **AI Development:** Claude Code
